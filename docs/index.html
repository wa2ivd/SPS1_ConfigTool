Connect SPS-1

Before continuing, please:

  • Connect a single SPS-1 to the serial port.
  • Ensure input power is connected to the SPS-1.
  • Only one SPS-1 may be connected — this tool uses broadcast messages and will misbehave if multiple devices respond.

When you click Connect, your browser will show a list of available serial ports. Pick the one the SPS-1 is attached to.

Browser Not Supported

This tool uses the Web Serial API, which is only available in Chromium-based browsers.

Please open this page in:

  • Google Chrome (89 or newer)
  • Microsoft Edge (89 or newer)
  • Brave, Opera, or another Chromium-based browser

Firefox and Safari do not support direct serial-port access from web pages.

Switch is ON

The SPS-1 output power switch is currently ON. Configuration parameters can only be modified while the switch is in the IDLE state (both DCN Enable and Local Enable OFF).

Turn the switch OFF (set local enable and/or remove the DCN enable command) to continue. This window will close automatically when the switch returns to idle.
